Calls

The Calls module in BostelCRM allows Users to schedule and log a record of inbound and outbound calls that they may be a participant of. You can access the Calls module from the Module List menu on the top navigation menu or from the Calls button on the Bottom Navigation menu.

Calls Actions

Depending on the screen or device size, you can access the Calls actions either from the Calls module menu dropdown or via the Sidebar.

  • Log Call

    A new form is opened in Edit View to allow you to create a new Call record.

  • View Calls

    By clicking on it, you will be redirected to the List View for the calls module, which enables you to search and display call records.

Managing Calls

  • To sort the Calls List View, click on any sortable column title. This will sort the column in ascending or descending order.
  • If you need to search for a specific call, refer to the Search section of this user guide.
  • To access the details of a call, click on its subject in the List View. This will open the call record in Detail View.
  • If you want to edit a call's details, you have two options: in Detail View, click Edit on the Actions menu, or click the pencil/edit button in List View, make the necessary changes, and click Save.
  • If you need to create a duplicate of a call, go to Detail View and click Duplicate on the Actions menu. Then make the desired modifications to the new record and save it.
  • To mark a call as held, open its Detail View and click Close on the Actions menu. This will change the call status to hold. Alternatively, you can click Close and Create New to close the current call and create a new one with the same subject in Edit View.
  • If you need to reschedule a call, click the Reschedule button on the Call Detail View. For a more detailed guide on rescheduling calls, refer to the Reschedule section of this user guide.
  • To remove a call from the list, open its Detail View and select Delete from the Actions menu.
  • To track all changes to audited fields in a call record, click the View Change Log button on the Call Detail View or Edit View.
